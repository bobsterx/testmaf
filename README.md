# Mafia Telegram Bot

Телеграм-бот для проведення гри «Мафія» в групових чатах. Підтримує людських гравців і до шести ботів, автоматичну роздачу ролей та цикли «ніч/день/голосування» з таймерами.

## Запуск

1. Встановіть залежності:

```bash
pip install -r requirements.txt
```

2. Запустіть бота:

```bash
python -m mafia_bot.bot
```

Бот уже містить токен із завдання. Після запуску додайте його до групи, усі гравці повинні написати йому `/start` у приваті.

## Команди

- `/start` (у приваті) — активує діалог із ботом для надсилання ролі.
- `/mafia` (у групі) — створює лобі з кнопками реєстрації, додавання ботів та старту гри.

Після старту рольові дії виконуються через кнопки в особистих повідомленнях. Таймери: ніч — 60 с, день — 60 с, голосування — 30 с. Боти приймають рішення автоматично, і в чаті з’являються атмосферні логи та гіфки з папки `gifs`.
